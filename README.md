Command with Next.js:

```bash
# Development
npm run dev
# Build
npm run build
# Start the compiled version
npm run start
```

Command with json-server:

```bash
# Start
npx json-server db.json --port={port}
```

Technology stack:

- Next.js
- Typescript
- EsLint
- Zustand v5
- Next-Auth (Google and Credentials)
- SSG and IRS
- Image Optimization
- JSON-server (No more than version 0.17.4)
- Server Actions
